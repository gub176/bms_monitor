BMS 监控系统 - 开发服务器测试

✅ 服务状态检查

📋 测试步骤

  1. ✅ 开发服务器已启动在 http://localhost:5176
  2. ⚠️ 请在无痕/隐私模式中打开 http://localhost:5176
  3. ⚠️ 如果仍有问题，检查浏览器控制台 (F12)

🔧 已知问题

SES 扩展冲突: 某些浏览器扩展（如密码管理器）会注入 SES (Secure ECMAScript) 导致 JavaScript 执行失败。

解决方案

  • 方案 1: 使用浏览器无痕/隐私模式
  • 方案 2: 暂时禁用所有浏览器扩展
  • 方案 3: 换用其他浏览器（如 Edge、Chrome、Firefox）
