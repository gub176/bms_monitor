## 1. 系统简介

### 1.1 产品概述

BMS 监控系统是一个面向企业级用户的电池管理系统（Battery Management System）监控平台，提供实时的设备状态监控、告警管理、数据分析等功能。

### 1.2 核心功能

| 功能模块 | 说明 |
|----------|------|
| 仪表盘 | 实时查看设备总数、在线状态、活跃告警、系统健康度 |
| 告警中心 | 管理和查看所有系统告警，支持筛选、搜索、批量操作 |
| 设备详情 | 查看单个设备的电芯电压、温度、SOC/SOH 等详细数据 |
| 设备绑定 | 绑定新的 BMS 设备到监控系统 |

### 1.3 系统要求

- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **网络**: 稳定的互联网连接
- **屏幕分辨率**: 推荐 1280x720 或更高

## 4. 仪表盘

### 4.1 功能概述

仪表盘是系统的主页面，提供设备运行状态的概览。

### 4.2 核心指标卡片

```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│  ⚡ 设备总数 │  📶 在线设备 │  🔔 活跃告警 │  📊 健康度  │
│    12 台     │     8 台     │     3 条     │    95%     │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

| 指标 | 说明 |
|------|------|
| 设备总数 | 已绑定的 BMS 设备总数 |
| 在线设备 | 当前在线（5 分钟内有数据上报）的设备数 |
| 活跃告警 | 当前未恢复的告警数量 |
| 系统健康度 | 在线设备占比百分比 |

### 4.3 设备列表

设备列表显示所有已绑定的设备信息：

| 列名 | 说明 |
|------|------|
| 设备名称 | 设备制造商名称或自定义名称 |
| 状态 | 在线/离线状态 |
| 电压 | 实时电压（伏特 V） |
| 温度 | 实时温度（摄氏度°C） |
| SOC | 电池荷电状态（百分比%） |
| 电芯数 | 电池串数 |

### 4.4 操作说明

- **点击设备行**：进入该设备的详情页面
- **绑定新设备**：点击右上角"绑定新设备"按钮添加设备

### 4.5 空状态说明

当没有设备时，系统显示：

```
┌─────────────────────────────────────┐
│            ⚡                       │
│           暂无设备                  │
│  绑定 BMS 设备后即可实时监控电压、     │
│    温度、SOC 等关键数据              │
│                                     │
│      [+ 绑定设备]                   │
└─────────────────────────────────────┘
```

## 5. 告警中心

### 5.1 功能概述

告警中心用于实时监控系统告警，支持查看活动告警和历史记录。

### 5.2 统计卡片

```
┌─────────────┬─────────────┬─────────────┬─────────────┐
│ 🔥 活动告警 │ 📜 历史告警 │ ⚠️ 严重告警 │ 📢 今日告警 │
│     3 条    │    15 条     │     1 条     │     8 条    │
└─────────────┴─────────────┴─────────────┴─────────────┘
```

### 5.3 告警级别

| 级别 | 图标 | 说明 |
|------|------|------|
| 严重 | 🔴 | 需要立即处理的紧急情况 |
| 警告 | 🟠 | 需要关注的异常情况 |
| 提示 | 🔵 | 一般性提示信息 |

### 5.4 告警类型

| 告警类型 | 说明 |
|----------|------|
| 短路 | 检测到电路短路 |
| 欠压 | 电压低于安全阈值 |
| 过压 | 电压高于安全阈值 |
| 电芯电压过高 | 单个电芯电压过高 |
| 电芯电压过低 | 单个电芯电压过低 |
| 温度过低 | 温度低于工作范围 |

### 5.5 筛选与搜索

#### 按级别筛选
- 点击级别标签（严重/警告/提示）进行筛选
- 可多选
- 点击"重置"清除筛选条件

#### 按设备筛选
- 选择特定设备查看其告警
- 选择"全部设备"显示所有告警

#### 按时间筛选
- 选择日期范围筛选告警

#### 关键词搜索
- 输入设备 ID 或告警描述进行搜索

### 5.6 批量操作

#### 标记已读

1. 勾选需要标记的告警（左侧复选框）
2. 点击"标记已读"按钮
3. 系统提示"已标记 X 条告警为已读"

### 5.7 导出告警

1. 点击右上角"导出 CSV"按钮
2. 系统将当前筛选条件下的告警导出为 CSV 文件
3. 文件自动下载到本地

### 5.8 活动告警 vs 历史记录

系统使用 Tab 页签分离两种告警：

| Tab | 说明 |
|-----|------|
| 活动告警 | 当前未恢复、正在发生的告警 |
| 历史记录 | 已恢复的历史告警记录 |

## 6. 设备详情

### 6.1 访问方式

- 在仪表盘设备列表中点击任意设备行

### 6.2 设备信息卡片

```
┌─────────────────────────────────────────────────────────┐
│  ⚡ BMS 设备 A         [● 在线]                          │
│  device-abc123...      2 个告警                          │
│                        [正常运行] [充电中] [已并网]      │
└─────────────────────────────────────────────────────────┘
```

### 6.3 关键参数

| 参数 | 说明 |
|------|------|
| SOC | 荷电状态（0-100%） |
| 总电压 | 电池组总电压（V） |
| 总电流 | 实时电流（A） |
| SOH | 健康状态（0-100%） |
| 温度范围 | 最高/最低温度（°C） |

### 6.4 功率信息

| 指标 | 说明 |
|------|------|
| 充电功率 | 当前充电功率（kW） |
| 放电功率 | 当前放电功率（kW） |

### 6.5 电芯详情

电芯详情表格显示每个电芯的实时数据：

| 列名 | 说明 |
|------|------|
| 序号 | 电芯编号（#01, #02, ...） |
| 电压 | 单体电压（V） |
| SOC | 单体荷电状态（%） |
| 温度 | 单体温度（°C） |

### 6.6 温度状态图例

```
温度状态：● 正常 (≤45°C)  ● 警告 (45-60°C)  ● 严重 (>60°C)
```

## 7. 常见问题

### 7.1 登录问题

**Q: 忘记密码怎么办？**  
A: 请联系系统管理员重置密码。

**Q: 登录时提示"网络错误"？**  
A: 请检查网络连接，或尝试刷新页面。

### 7.2 设备问题

**Q: 设备显示离线？**  
A: 设备超过 5 分钟未上报数据将显示离线，请检查设备网络连接。

**Q: 如何绑定新设备？**  
A: 在仪表盘点击"绑定新设备"按钮，按提示操作。

### 7.3 告警问题

**Q: 告警如何恢复？**  
A: 当导致告警的条件消失后，告警会自动标记为"已恢复"。

**Q: 能否关闭告警通知？**  
A: 请联系管理员配置告警通知设置。

### 7.4 浏览器兼容性

**Q: 系统在某些浏览器显示异常？**  
A: 建议使用最新版本的 Chrome、Firefox、Safari 或 Edge 浏览器。
